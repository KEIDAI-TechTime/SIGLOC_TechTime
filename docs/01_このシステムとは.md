# このシステムとは

## このシステムを一言でいうと

> **SIGLOC-onlineプログラムの「準備作業」と「報告書作成」を自動化するシステム**です。

---

## なぜこのシステムが必要なのか

### SIGLOC-onlineの運営は大変

SIGLOC-onlineは、世界中の学生が参加する国際オンライン教育プログラムです。
このプログラムを開催するたびに、以下のような**大量の準備作業**が発生します。

```
【毎回の開催に必要な準備作業】

  ┌─────────────────────────────────────────┐
  │  ① フォルダをたくさん作成する          │
  │    （学生用、教員用、システム用など）    │
  └─────────────────────────────────────────┘
                    ↓
  ┌─────────────────────────────────────────┐
  │  ② ファイルを前回分からコピーする      │
  │    （設定ファイル、テンプレートなど）    │
  └─────────────────────────────────────────┘
                    ↓
  ┌─────────────────────────────────────────┐
  │  ③ Googleフォームを更新する            │
  │    （応募、入学、アンケートなど）        │
  └─────────────────────────────────────────┘
                    ↓
  ┌─────────────────────────────────────────┐
  │  ④ 日付やリンクを全部書き換える        │
  │    （チェックリスト、マニュアルなど）    │
  └─────────────────────────────────────────┘
                    ↓
  ┌─────────────────────────────────────────┐
  │  ⑤ 修了証のテンプレートを作成する      │
  └─────────────────────────────────────────┘
```

これらを**すべて手作業**で行うと、何時間もかかり、ミスも起こりやすくなります。

---

## このシステムが解決すること

### ビフォー・アフター

| | 手作業の場合 | システムを使う場合 |
|---|---|---|
| **作業時間** | 数時間〜1日 | 数十分 |
| **ミスの可能性** | 高い（リンク切れ、日付間違いなど） | 低い（自動チェック機能付き） |
| **必要なスキル** | Google Driveの操作に習熟が必要 | ボタンを押すだけ |

---

## 2つのシステム

このフォルダには**2つのシステム**の資料が含まれています。

### システム1: 新期開始時サポートシステム

**いつ使う？** → 新しいSIGLOC-onlineを開催する準備をするとき

**何をしてくれる？**
- フォルダを自動で作成
- ファイルを自動でコピー
- Googleフォームを自動で更新
- リンクや日付を自動で書き換え
- 作成結果を自動でチェック

```
【イメージ】

  前回のSIGLOC          このシステム           今回のSIGLOC
  ┌──────────┐         ┌──────────┐          ┌──────────┐
  │ 20th回   │   →    │  ボタン   │    →    │ 21st回   │
  │ のフォルダ │        │  ポチッ   │          │ のフォルダ │
  └──────────┘         └──────────┘          └──────────┘
                           │
                     自動でコピー＆
                     日付などを更新！
```

---

### システム2: 事後レポート作成システム

**いつ使う？** → SIGLOC-online終了後、アンケート結果をまとめるとき

**何をしてくれる？**
- アンケート回答を自動で集計
- グラフを自動で作成
- 日本語版・英語版のレポートを自動で生成
- 英語の回答を日本語に自動翻訳

```
【イメージ】

  アンケート結果        このシステム              レポート
  ┌──────────┐         ┌──────────┐          ┌──────────┐
  │  回答    │   →    │  ボタン   │    →    │ 日本語版  │
  │  データ  │         │  ポチッ   │          │ 英語版    │
  └──────────┘         └──────────┘          └──────────┘
                           │
                     自動で集計＆
                     グラフ作成！
```

---

## どこで動いているのか

このシステムは、すべて**Googleの中**で動いています。

```
┌─────────────────────────────────────────────────────────┐
│                    Google Drive                         │
│                                                         │
│   ┌─────────────┐     ┌─────────────┐                  │
│   │  Excel風の  │ ──▶ │  プログラム  │                  │
│   │   画面     │     │  (自動処理)  │                  │
│   │            │     │             │                  │
│   │ ボタンを   │     │ フォルダ作成 │                  │
│   │ クリック   │     │ ファイルコピー│                  │
│   └─────────────┘     │ 日付更新    │                  │
│                       └─────────────┘                  │
│                             │                          │
│                             ▼                          │
│                    ┌─────────────────┐                 │
│                    │   完成！        │                 │
│                    │ 新しいSIGLOC用  │                 │
│                    │ フォルダ＆ファイル│                │
│                    └─────────────────┘                 │
└─────────────────────────────────────────────────────────┘
```

**ポイント**
- 特別なソフトをインストールする必要はありません
- インターネットにつながっていれば、どこからでも使えます
- Googleアカウントでログインするだけで使えます
