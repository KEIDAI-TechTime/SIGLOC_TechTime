# SIGLOC-onlineプログラムについて

## SIGLOCとは

**SIGLOC**（シグロック）は、「**S**ocial **I**nnovation for **G**lobal **LOC**al sustainability」の略で、日本語にすると「**地球規模の持続可能性のための社会的イノベーション**」という意味です。

大阪公立大学の**COIL事業部門**が運営する、国際オンライン教育プログラムです。

---

## どんなプログラム？

### 基本情報

| 項目 | 内容 |
|------|------|
| **形式** | 完全オンライン（インターネットで参加） |
| **期間** | 約3週間 |
| **参加者** | 世界各国・地域の大学生 |
| **参加人数** | 約40〜50名（例：第15回は7か国・地域から45名） |
| **開催頻度** | 年に複数回（例：3月、7月など） |
| **テーマ** | SDGs（持続可能な開発目標）に関連した社会課題 |

---

## プログラムの流れ

```
【3週間のプログラム構成】

Week 1                    Week 2                    Week 3
┌─────────────┐          ┌─────────────┐          ┌─────────────┐
│ セッション1  │    →    │ セッション2  │    →    │ セッション3  │
│             │          │             │          │             │
│ 非同期      │          │ 地域社会と   │          │ 地球規模の   │
│ ディベート   │          │ 社会問題    │          │ 持続可能性   │
└─────────────┘          └─────────────┘          └─────────────┘
       ↑                        ↑                        ↑
 オンラインで              グループで              最終発表
 意見交換                  課題を議論              プレゼン
```

### 各セッションの内容

| セッション | 内容 | 活動例 |
|-----------|------|-------|
| **セッション1** | 非同期ディベート | 世界中の参加者とオンラインで議論 |
| **セッション2** | 地域社会と社会問題 | 自分の地域の社会課題を調査・発表 |
| **セッション3** | 地球規模の持続可能性 | チームで解決策を考え、発表 |

### 文化交流活動

プログラム中は、学習以外にも文化交流の機会があります。

- **マイワンスライド**: 自己紹介スライドを作成して共有
- **写真ショーケース**: 自分の国や地域の写真を共有
- **クイズ**: 異文化理解のためのクイズ大会
- **グループ別文化活動**: 小グループでの交流イベント

## このシステムとの関係

上記の「開催前の準備」と「開催後のレポート作成」を**自動化**するのが、このフォルダに含まれるシステムです。

| 作業 | 対応するシステム |
|------|-----------------|
| 開催前の準備（フォルダ、フォーム、テンプレート作成） | **新期開始時サポートシステム** |
| 開催後のレポート作成 | **事後レポート作成システム** |

詳しくは → [03_システムでできること.md](./03_システムでできること.md)
