- **今回の 修了証 関連フォルダを検索**

  - **今回の Online フォルダを検索し、IDを取得。**

  - **今回の SYSTEMフォルダを検索し、IDを取得。**

  - **今回の** 修了証**フォルダを検索し、IDを取得。**

  1.  **確認の実施**

- **システム設定ファイルの取得**

  - 今回システム設定ファイルを取得。

  - 今回のシステム設定シートを取得。

- **修了証ファイルの確認**

  - 修了証ファイルを取得

  - システム設定ファイルの修了証ファイル ID を検証

- **修了証ファイルのシート名確認**

  - 取得した修了証ファイルを開き、シート名をシステム設定と照合

- **修了証原本フォルダの確認**

  - 修了証原本のフォルダ ID を取得し、システム設定と照合

- **Certificate of Completion フォルダの確認**

  - Certificate of Completionのフォルダ ID を取得し、システム設定と照合
