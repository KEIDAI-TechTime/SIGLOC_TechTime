###  **処理の流れ**

- **初期設定**

### **ログの設定**

- ログを記録するために **settingLog** を呼び出す。

### **チェックリストの設定**

- テストを記録するために **settingTest** を呼び出す。

<!-- -->

- **引数のチェック**

  - **checkFolderId**で今回のフォルダIDを検証。

  - **checkString** で今回の開催回情報を検証。

  - **checkFolderId** で前回のフォルダIDを検証。

  - **checkString** で前回の開催回情報を検証。

<!-- -->

- **今回のフォルダを検索**

  - 今回の Online フォルダを検索し、IDを取得。

  - 今回の Google_Formフォルダを検索し、IDを取得。

  <!-- -->

  - **確認の実施**

- **システム設定ファイルの取得**

  - 今回システム設定ファイルを取得。

  - 今回のシステム設定シートを取得。

- **Gmail登録フォーム(回答)ファイルの確認**

  - システム設定ファイルの**登録フォーム(回答)**ファイル ID を検証

- **Gmail登録フォーム(回答)ファイルのシート名確認**

  - 取得した**登録フォーム(回答)**ファイルを開き、シート名をシステム設定と照合

- **応募フォームファイルの確認**

  - システム設定ファイルの**登録フォーム**ファイルを検証

- **応募フォーム(回答)ファイルのシート名確認**

  - 取得した**応募フォーム(回答)**ファイルを開き、シート名をシステム設定と照合

  - **応募フォーム**のリンクが**応募フォーム(回答)**のIDと同一か確認

- **入学フォームファイルの確認**

  - システム設定ファイルの**登録フォーム**ファイルを検証

- **入学フォーム(回答)ファイルのシート名確認**

  - 取得した**入学フォーム(回答)**ファイルを開き、シート名をシステム設定と照合

  - **入学フォーム**のリンクが**入学フォーム(回答)**のIDと同一か確認

- **事後報告フォームファイルの確認**

  - システム設定ファイルの**登録フォーム**ファイルを検証

- **事後報告フォーム(回答)ファイルのシート名確認**

  - 取得した**事後報告フォーム(回答)**ファイルを開き、シート名をシステム設定と照合

  - **事後報告フォーム**のリンクが**事後報告フォーム(回答)**のIDと同一か確認

####  **フォルダIDの返却**

- 作成した thisTimeFolderId を返すことで、処理の結果を次の工程に渡す。
